# MDU25 eksamen: Spilcaféen
### December 2025 af Marie-Luise Kuppe
Denne opgave består af optimering af Spilcaféen temaet, hvor både design og hjemmeside er blevet optimeret.

## Kode optimering

* Ændring af HTML for at tilpasse det nye design.
    * `<header>` er blevet optimeret med både logo og menu sektion.
    * `<main>` sektion er blevet mere opdelt, for bedre styling.
    * Små ændringer på `filterbar`.
    * `dialog-id` er blevet implementeret fremfor `onclick`.

* Ændring af CSS for at tilpasse det nye design.
    * Oprydning i gammel kode.
    * Tilføjet styling for `header`.
    * Ændret styling på `filterbar`, modal (`game dialog`), og på `game.list` + `game.cards` og andre små ting.

* Ændring af JavaScript for at tilpasse det nye design
    * Tilføjet filter dropdowns, herunder Sværhedsgrad og Flere filter; samt fungerende dropdowns i Flere filter, såsom Alder, Sprog, Lokation og Rating.
    * Tilføjet mere indhold til modal for bedre brugeroplevelse.

### Validering
Både HTML og CSS er blevet valideret - og viser ingen errors / warnings.
